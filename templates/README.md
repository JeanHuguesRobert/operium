# Templates

This directory contains starter files for Operium registries.

Initial templates should cover:

- hosts;
- repositories;
- services;
- domains;
- health records;
- evolutions;
- decisions.

Templates should remain simple, human-readable and usable before scripts or dashboards are introduced.
