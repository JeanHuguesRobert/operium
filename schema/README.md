# Schema

This directory will contain generic YAML schemas for Operium registries.

Initial schema targets:

- `host.schema.yaml`
- `repository.schema.yaml`
- `service.schema.yaml`
- `domain.schema.yaml`
- `health-check.schema.yaml`
- `evolution.schema.yaml`
- `decision.schema.yaml`

Schemas are intentionally not added before the first templates stabilize.
